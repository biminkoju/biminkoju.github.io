Some Basic Terms Related to the Internet

Search engine

A search engine is a software system that is designed to carry out web searches(internet searches). it is a software searches a databaseof information according to the user's query. It provies a list of results that best match with what the user is trying to find. Today there are many searchengines avialabe on the internet, each with its own abilities and features. google, yahoo, bing, ask, etc. are some examples of search engines.

Web Browser

Web browser is a program to navigate the internet. Safari, internet explorer, Mozilla forefox, opera, netscape navigator are some web browsers. witht eh help of webbrowser, we can access various kinds of information, articles, graphics, and audio-visuals as per our need.

Website

A website refers to a central location that contains more than one web page or a series of web pages. for example, https://google.com, https://archlinux.org, https://youtube.com are links to websites. which contains thousand of diffrent web pages

Homepage

Homepage is the introductory page of a website, typically serving as a table of contents for the site. it is the main web page of a website. The term can also refer to one or more pages always shown in a web browser when the application starts up.

uniform resource locator(URL)

Uniform Resource Locator(URL) is the adress that defines the route to a file on an internet server(web server or mail server). Urls are typed into a webbrowser to acess the webpage and files. URls are embeded within the pages themselves as links. the url is specific source in the internet. for example, https://google.com, is an url where 'www' is file server, 'google' is domain name and '.com' is top domain name(tdn)

Domain Name Server(DNS)

The domain name system(dns) is a hierarchical and decentralized naming system for computers, services, or other resources connected to the internet or a private network. it associates various information with domain names assigned to each of the participating entities, the main purpose of dns server is to conver domain name to IP adresses and IP adresses to domain name. every domain name has its own IP adress wich is not easy to remember rather than name. Therefore, the user type domain names instead of IP adresses. by providing a worldwide, distributed directory service. The domain name system as been an essential componet of the functionality of the internet since 1985
