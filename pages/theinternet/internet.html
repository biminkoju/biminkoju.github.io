Introduction

The Internet is a vast 'interconnection of computer networks' that spans the globe.
It is the largest computer network system in the world which many computers are interconnected to peform various tasks. It interconnects LAN, MAN, WAN, and various other networks through TCP/IP.
Therefore the internet is the global network of the computer networks connecting the millions of computers. Millions of computers around the world can share information at the same time by using this worldwide network system.
In this way, it is a computer netowrk consisting od a worldwide network of computer networks that use the TCP/IP network protocol to facilitate data transmission and exchange.
